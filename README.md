# Campus Event Management Platform

A prototype Campus Event Management Platform for managing events, student registrations, attendance, and feedback collection.

---

## Features

### Admin Portal
- Create, update, and cancel events
- View student registrations and attendance
- Collect feedback and generate reports

### Student App
- Browse events by type or date
- Register for events
- Check-in for attendance (QR/manual)
- Submit feedback (rating 1â€“5)

### Reports
- Total registrations per event
- Attendance percentage
- Average feedback score
- Top active students

---

## Tech Stack

- **Backend:** Python, Flask, SQLAlchemy
- **Database:** SQLite (for prototype) / PostgreSQL 
- **Frontend:** Minimal HTML/CSS/JS (can be replaced with React/Vue)
- **Version Control:** Git / GitHub
